# 口の中心と開き具合を検出

## 概要
口の中心と開き具合を検出し、canvasに描画する

## フォルダ
```
├── README.md
├── assets
├── index.html
├── package.json
└── yarn.lock
```

## 使い方
viteを使用しています。

### install
```
$ yarn 
```

### 開発サーバー起動
```
$ yarn dev
yarn run v1.22.19
$ vite --host
  VITE v4.4.9  ready in 254 ms
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://172.21.214.107:5173/
```
